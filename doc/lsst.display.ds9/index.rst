################
lsst.display.ds9
################

``lsst.display.ds9`` package provides an interface between the `SAOImage DS9`_ image display tool and LSST objects.

.. _SAOImage DS9: http://ds9.si.edu/

Contributing
============

``lsst.display.ds9`` is developed at https://github.com/lsst/display_ds9.
You can find Jira issues for this module under the `display_ds9 <https://jira.lsstcorp.org/issues/?jql=project%20%3D%20DM%20AND%20component%20%3D%20display_ds9>`_ component.
